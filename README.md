# Bullet-Cam

AR face filter that tracks your face in real time and overlays the Gettysburg College bullet mascot on your head using your webcam. Open your mouth and it shows GO BULLETS!

Currently supports up to 2 faces at once, can be scaled to more.

## Built With
- Python
- OpenCV
- MediaPipe
- NumPy
